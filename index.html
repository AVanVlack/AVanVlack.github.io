Hello, my name is Andrew. I am a web and software developer living in Portland, Oregon. My passion is creating complex and intricate, but elegant software that is hidden away behind a clean user focused interface to give others new tools and experiences. This site is currently undergoing construction so please come back soon.

Twitter Dashboard for Chromecast

This project is an experiment with both of the use of a back end framework and the use of a television with chromecast. The twitter data is pulled by the node.js backend and then socket.io is used to send the filtered tweets to the chromecast display. Front end controls are all handled by the chromecasts databus.
